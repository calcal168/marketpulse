# MarketPulse

MarketPulse is a React Native / Expo news app that brings market-moving headlines into one mobile feed. It organizes news by source and category, supports saved articles, reads headlines aloud, opens full articles in an in-app reader, and translates headlines/articles into Chinese.

## Features

- News feed from multiple RSS sources
- Categories: Political, Financial, Economic, People, Health, and Tech
- Source filters for Yahoo, BBC, BBC Chinese, CNA, Sina, Al Jazeera, and Google News
- Text-to-speech for headlines and full articles
- Slower read-aloud speed for easier listening
- In-app article reader
- Chinese translation for headlines, summaries, and article text
- Save favorite articles
- Keyword alerts for important market terms
- Market-impact brief generation

## Tech Stack

- Expo
- React Native
- TypeScript
- Expo Router
- Expo Speech
- React Native WebView
- AsyncStorage
- RSS/XML parsing with fast-xml-parser
